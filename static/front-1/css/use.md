# 常用 css 样式大全

## 修改 placeholder 的颜色
```css
input::-webkit-input-placeholder, textarea::-webkit-input-placeholder { 
    color: #666; 
} 
input:-moz-placeholder, textarea:-moz-placeholder { 
    color: #666; 
} 
input::-moz-placeholder, textarea::-moz-placeholder { 
    color: #666; 
} 
input:-ms-input-placeholder, textarea:-ms-input-placeholder { 
    color: #666; 
} 
```
