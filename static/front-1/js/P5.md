## 继承

原型继承的问题。

1. constructor指向
2. 如果父类的实例属性为引用类型，则会相互影响
3. 共享属性不能在实例化子类的时候不能通过参数

借用构造函数的问题 `Person.call(this)`
1. 只能继承属性
2. 方法如果继承，会不共享

组合继承的问题
1. 会调用2个构造函数
2. 父实例的属性没有用到

最佳实践
- 做一个中间对象，方法都继承这个中间对象。
- 属性通过借用构造函数方式继承

```
function extendFn(child, parent){
    var proto = Object.create(parent.prototype)
    child.prototype = proto
    proto.constructor = child
}
```
